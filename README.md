# AI Architect Accelerator

**Author:** Corina Guzman

A 16-week, project-based journey from Senior Software Engineer to AI Architect.

---

## Mission

Build a deep understanding of Artificial Intelligence by combining theory, mathematics, software engineering, and hands-on projects.

Rather than learning isolated tools, this repository documents the process of learning how to design, build, evaluate, and explain production-quality AI systems.

---

## Learning Philosophy

This repository follows five principles:

1. Understand concepts before memorizing formulas.
2. Connect every concept to a real-world engineering problem.
3. Build something every week.
4. Learn by questioning assumptions and exploring tradeoffs.
5. Think like an AI Architect, not just an AI user.
