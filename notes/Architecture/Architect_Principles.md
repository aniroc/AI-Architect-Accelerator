## AP-001

Better data usually beats a better algorithm.

## AP-002

Every prediction is only as good as the evidence available.

## AP-003

Models learn patterns, not truth.

## AP-004

Correlation does not imply causation.

## AP-005

Every AI system makes assumptions.

## AP-006

Don't confuse the target with a feature.

## AP-007

Every feature tells a story. Before removing or transforming a feature, ask what real-world phenomenon it might represent.

## AP-008

Never assume the data is complete. Measure data quality before modeling.

## AP-009

Missing data does not necessarily describe missing events.

## AP-010

Derived features simplify information but may also discard useful detail.

## AP-011

A useful feature often predicts the outcome because it represents an underlying mechanism, not because it directly causes the outcome.
