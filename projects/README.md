# Project 01 - Titanic Data Exploration

## Objective

Learn the first phase of every Machine Learning project: understanding the data before building a model.

## Business Problem

Given passenger information, determine which factors appear to influence survival aboard the Titanic.

This project focuses on exploration and hypothesis generation rather than prediction.

## Learning Objectives

* Load data using pandas
* Explore a dataset
* Identify features and labels
* Distinguish numerical and categorical variables
* Detect missing data
* Form hypotheses before modeling

## Dataset

The Titanic dataset is loaded directly from the Seaborn library.

* Feature analysis
